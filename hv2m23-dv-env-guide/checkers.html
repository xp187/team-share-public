各链细节

data_merge（data_merge_scoreboard.sv）

  • monitor 采 CENTER1.PIX_DATA 写成 ppm；golden 从 {SIM_TEST_PATH}/outResult/data_merge/frame_N_data_mergeM.txt 读。
  • 逐字节比对，不等报 uvm_error("data merge error") 并注明帧/行/像素/RGB。

digital_top（digital_top_scoreboard.sv）

  • 比对 DBR/DBL 左右 odd/even 四路，golden 为 outResult/digital/frame_N_digitalMchesel_{Lodd,Leven,Rodd,Reven}.txt。
  • DPLC_MODE 时改读 dplc.pl 输出的 DPLC_frameN_idM.ppm。

analog（analog_data_output_scoreboard.sv）

  • 采 SIM_OUT[14469:10] 模拟输出，4 个 analysis imp（monitor/pol/vbk/unlock），按帧比像素、极性、VBK、chopper。
  • Disable channel 行为模型期望为 10'h22（HiZ）。

chopper（chop_monitor / chop_scoreboard）

  • 根据 SD_CHOP[3:0] 与 G_CHOP[3:0] 产生期望 chopper 序列，与 analog 中分离出的 chop bit 比对。

drdod_input / drdod_output

  • drdod_process.py 生成 golden：检测输入 pattern 类别（OD_k）、面板类型（DRD_PANEL）、1pto2p 交叉重整等。
  • drdod_input 检查 pattern 检测与加权选择；drdod_output 检查最终输出数据与 OD_time。

checksum 说明

  • HV2M23 无独立 checksum checker（不同于 HK1V11）。寄存器装载正确性由 datapath 用例和 I2C 读写用例间接覆盖。

辅助工具

工具 / 脚本作用
script/run_tc.sh运行单个 testcase，支持 TOP / TOP_COV / TOP_NETLIST / FPGA_TOP / verdi / cov / cov_rpt。
script/check.py遍历仿真目录，检查 compile.log 中的 "Error" 与 sim.log 中的 "UVM_ERROR"，输出 compile_result.txt / compare_result.txt。HV2M23 使用 check.py，不是 HK1V11 的 check_sim_results.sh。
script/merge_coverage.py合并所有用例的 simv.vdb 生成统一覆盖率报告；注意其中可能存在硬编码的项目路径，使用前需确认。
script/produce_checker1.pychecker 生成辅助脚本。
script/user_def_addon.pluser_def.sv 附加定义生成。
top/tests/create_new_case.sh从模板创建新用例。
top/tests/run_case.py回归逐 case 运行，可接受用例名或 case_list.txt。
top/tests/run_parallel.py并发回归运行。
top/tests/regression.py / regression_cov.py回归调度与覆盖率回归脚本。
top/tests/produce_isp_register*.py / produce_all_register.py / user_def_register*.py寄存器配置 / user_def 生成辅助。
top/tests/copy_test.sh / produce.sh / creat_cases.sh用例复制、产物生成、批量建case 等辅助脚本。

开关与判决

开关说明
env_cfg.data_merge_check_ongate data_merge scoreboard
env_cfg.digital_top_check_ongate digital_top scoreboard
env_cfg.analog_check_ongate analog scoreboard
env_cfg.chopper_check_ongate chopper scoreboard
env_cfg.drdod_input_check_ongate drdod_input scoreboard
env_cfg.drdod_output_check_ongate drdod_output scoreboard

所有 scoreboard 的 uvm_error 汇聚到 report server；base_test.report_phase 统计 err_num，为 0 则打印 TEST CASE PASSED，否则 FAILED。
