用例结构与新建

用例目录解剖（以 t_8b1lane 为例）

文件作用
t_8b1lane.svtest class：创建 isprx_env_cfg，填入 user_def.sv 宏（HACT/HBP/VTOTAL/FPS/PAIR_NUM/FRAME_NUM/CHIP_SEL/DV…），设 6 个 *_check_on 开关，config_db 下发 env_cfg 和 default_sequence
user_def.sv本例全部时序/分辨率宏
cfg_frame0~8.txt每帧寄存器值（如 CHSEL 001、POLC 0、DOTC 0，值为二进制字符串），由 env_cfg 打包进 CTRL_F/L
pattern/frame_0..7.ppm输入图像（P3 格式 PPM），常是指向 ppm_mix_*/ 的 symlink
test_lib.sv / test.finclude 本例 .sv / 编译文件列表（user_def + waves_dumper + incdir）
waves_dumper.sv用例级自定义波形抓取

各用例的 t_xxx_vseq extends base_vseq 通常只调 super.body() —— 用例差异全在 test class 配置和 cfg_frame/pattern 文件。

名字字段作用目标说明
H1920 / H1800 / H1712 / H1640 / H1600 / H1440、V16、FPS60/90/120/165、P4/P8、L1/L2user_def.sv时序与分辨率宏；PORT_NUM=4/8，PAIR_NUM=1/2
